# Base UI Template

React 19 + TypeScript + Vite 8 + Tailwind CSS v4 + shadcn/ui v4 starter template.

## Usage

```bash
pnpx degit lix42/base-ui-template#main my-project
cd my-project

pnpm install
pnpm dev
```

## Tech Stack

- **React 19** with **TypeScript 5.9** (strict mode)
- **Vite 8** as bundler
- **Tailwind CSS v4** (CSS-first config)
- **shadcn/ui v4** (radix-vega style, zinc base color)
- **React Compiler** enabled
- **Biome** for linting and formatting

## Scripts

- `pnpm dev` — start dev server with HMR
- `pnpm build` — type-check and bundle for production
- `pnpm lint` — run Biome check
- `pnpm lint:fix` — run Biome check with auto-fix
- `pnpm preview` — preview production build
